FixComponentType(Type, Component)

Replaces existingComponent type with desiredType ONLY if desiredType is not assignable from the existingComponent type. That means if desiredType is Item and existingComponent type is Weapon, this will do nothing. If both types are the same, this will do nothing. Otherwise, this will replace existingComponent with a desiredType component and inherit all possible values.
